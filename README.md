# FlakeScope Studio

FlakeScope Studio 是一个 AI Flaky Test 诊断与稳定化工作台，面向前端、后端和端到端测试体系。

## 项目能力
- 读取 CI 重试日志、测试历史、Playwright trace、Jest/Vitest 输出、随机种子、mock 数据和相关源码
- 定位竞态、共享状态、时间依赖、随机种子、不稳定选择器和并发 worker 冲突
- 生成隔离 fixture、确定性 mock、语义等待、回归测试和重复运行验证
- 输出 PR 级稳定性报告、验证矩阵和后续测试治理建议

## 部署方式
这是一个静态 HTML demo，可以直接部署到 GitHub Pages、Netlify、Vercel 或任意静态托管平台。
